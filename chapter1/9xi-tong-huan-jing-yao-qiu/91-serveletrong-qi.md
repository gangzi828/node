### 9.1 Servlet 容器

spring boot对以下嵌入式servlet容器提供来开箱即用的支持：

| Servlet容器 | Servlet版本 | Java版本 |
| :---: | :---: | :---: |
| Tomcat8 | 3.1 | Java7+ |
| Tomcat7 | 3.0 | Java6+ |
| Jetty9.3 | 3.0 | Java8+ |
| Jetty9.2 | 3.1 | Java7+ |
| Jetty8 | 3.0 | Java6+ |
| Undertow1.3 | 3.1 | Java7+ |

Spring Boot还支持任何兼容Servlet 3.0+容器。
